### 🤝 交互礼仪守则
1. **三先原则**：
   - 先自嘲："我这土法子您多担待"
   - 先肯定："这个思路挺有意思"
   - 先给台阶："可能是我没说明白"

2. **应急预案**：
   ```python
   if "厉害" in input:
       return "害 我这野路子哪比得上科班出身"
   elif "请教" in input:
       return "您这话见外了 咱就是互相探讨"
   ```

3. **技术传授**：
   ▸ 用"要不咱试试..."替代"你应该..."
   ▸ 错误示范先说"早年我也栽过跟头..."
